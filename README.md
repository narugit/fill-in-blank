## How to Run
1. Please setup.
    ```
    $ python --version
    Python 3.7.6

    $ pip install -r requirements.txt

    $ node -v
    v11.14.0
    
    $ cd frontend
    
    $ npm install
    
    $ ../backend
    
    $ npm install
    ```
1. Please launch two shell screens.

    ```
    In the first shell screen
    $ cd frontend
    
    $ npm run start


    In the other shell screen
    $ cd backend
    
    $ npm run start
    ```

1. Please access to [http://localhost:8080](http://localhost:8080).
